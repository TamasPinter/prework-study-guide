HTML

  • !DOCTYPE html begins each HTML document
  • The head element contains info about a webpage
  • Meta tags give info on webpage display and search engine info
  • Include lang=en and UTF=8 for character display info
  • The body section displays info that the user can see
  • HTML uses semantic code to help make sense of code function, use this method as much as possible
  • Header tags should be used in order 1-6 on display
  • A is an anchor point and href is the target location
  • A margin indicates how much space we want arount the outside of an element
  • A padding indicates how much space we want on the inside of an element

CSS

  • Declerations contain 2 pieces of data, CSS property and it's value
  • Use * as a wild card symbol, applies rules to all elements on page
  • DRY, Do Not Repeat Yourself code, combine selectors when declerations are the same
  • Use class selectors to specifically apply CSS rules to specific defined elements

Git

  • git status: check what branch we are working in
  • cd : change directory
  • ls : list directories in current location
  • git checkout -b : create a new branch and switch to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in the script.
  • JavaScript adds interactivity to websites.
  • Declares variables with "let" followed by variables.
  • JavaScript is case sensative, pay attention!
  • Variables can be different data types, string ' ', number, boolean (true/false), array [], object ->anything
  • /* text */ comments, anything in the middle will be ignore by code
  • Common variable operations -> +,-,*,/,=,===,!,!==
  • Very common conditional statements if...else
  • Each line ends with ;
  • If statements for single variables, Array (for) when using groups of data
  • Arrays cab hold multiple types of data.
  • Using single item in Array by index reference. Starts @ 0
